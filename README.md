# OrderFlow AI — рабочий прототип

Прототип RPA-бота для автоматизации обработки заказов малых интернет-магазинов
(команда: Владимирова В. В., Алейник В. В., Шихрагимов И. Р.).

## Как запустить локально (через VS Code)

1. **Установите Node.js** (если ещё не установлен): https://nodejs.org — версия 18 или новее.
   Проверить, что установлен: в терминале `node -v`.

2. **Откройте папку проекта в VS Code**:
   `Файл → Открыть папку…` → выберите папку `orderflow-app`.

3. **Откройте встроенный терминал** в VS Code: `Терминал → Новый терминал`.

4. **Установите зависимости** (один раз):
   ```
   npm install
   ```

5. **Запустите приложение**:
   ```
   npm run dev
   ```
   В терминале появится ссылка вида `http://localhost:5173` — откройте её в браузере
   (обычно открывается автоматически, либо Ctrl/Cmd + клик по ссылке в терминале).

Приложение перезагружается на лету при изменении кода — можно редактировать `src/App.jsx`
прямо в VS Code и сразу видеть результат в браузере.

## Как остановить

В терминале — `Ctrl + C`.

## Как собрать финальную версию для показа на защите (без VS Code и терминала каждый раз)

```
npm run build
```

Появится папка `dist/` — это готовый статичный сайт. Его можно:
- открыть локально (`npm run preview`),
- либо бесплатно захостить на Vercel / Netlify / GitHub Pages, перетащив папку `dist`
  в панель загрузки — и получить постоянную публичную ссылку для отправки другим людям.

## Структура проекта

```
orderflow-app/
├── src/
│   ├── App.jsx       — всё приложение (стор, движок бота, экраны)

├── vite.config.js

```

## Примечание про хранение данных

Данные (заказы, правила, журнал) сохраняются в `localStorage` браузера — то есть
между перезагрузками страницы, но только на этом устройстве и в этом браузере.
Кнопка «Сбросить демо-данные» в Настройках очищает всё и создаёт новый стартовый набор.
